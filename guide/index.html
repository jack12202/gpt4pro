付款后找不到激活卡怎么办？

回到购买页的订单查询入口，输入下单时填写的联系方式查询。因此联系方式不能过于简单，也不能随手乱填。仍无法查询时，保留付款记录后联系嘉哥 GPT 客服核对。

获取数据后看到一大串代码，正常吗？

正常。页面返回的是 JSON 格式的 Session 数据。不要尝试只复制其中某一行，应全选并完整复制，再粘贴到激活中心对应输入框。

自己无法提交，可以联系人工处理吗？

可以。优先由本人在激活页面内完成；如果自己确实充值不了，可以通过 GPT4.pro 公布的官方客服渠道联系嘉哥 GPT 人工处理。先核对客服身份和处理范围，不要把 Session JSON 发给其他第三方。

提交后还没有看到 Plus，应该重新买吗？

不要重复购买或连续提交。先刷新或重新登录，确认自己查看的是正确账号；仍未生效时保留订单、付款记录、激活状态和错误截图，进入充值失败排查或联系人工协助。

哪些情况不建议直接购买？

账号无法正常登录、已经扣款但 Plus 未显示、已有订阅状态不确定、想开 Pro、需要发票或对公时，建议先咨询确认，避免重复付款或选错方案。
